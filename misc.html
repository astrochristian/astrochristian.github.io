Miscellaneous Projects

Programming languages, games, and hobby projects

Here you can find various hobby projects of mine, implemented in JavaScript and HTML. These include the interpreters for the stack-based esoteric programming languages I designed: Fourier and Noether. Some of these projects use my other Github account.

Nonogram

JavaScript implementation of the puzzle game, Nonogram (or Picross)

Sudoku

JavaScript implementation of Sudoku. Note that puzzles are generated randomly so may not be solvable or have a unique solution.

FourIDE: An IDE for Fourier

  • Function reference
  • Byte counting of code
  • Permalinking

Noether: A Stack Based Fourier

  • Overloading of functions
  • String operations
  • Functions with practical use

Randomised Weekly astro-ph.CO

  • Designed for use with Will Handley's weekly "arXiv Scrolling Club"
  • Updates list of papers every six hours
  • Set up to change the random seed every day, allowing people to view the same list
  • MathJaX support to properly render paper titles

Randomised Weekly astro-ph

  • General astronomy version of the cosmology "arXiv Scrolling Club"
  • Full list of astro-ph subarXivs that can be filtered
